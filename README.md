## Xnapper 的核心功能逐个拆解

### 自动背景与居中平衡

这个功能叫 Auto Balance。截图之后，Xnapper 会识别图中的有效内容，自动将主体居中放置在背景上。边缘颜色也会自适应扩展，不会出现奇怪的白边。

背景可以选渐变色、纯色，也可以用自定义图片。调节边框圆角、阴影强度、内边距大小，这些都有滑块可以拖动，所见即所得。

### OCR 文本识别

Xnapper 的文字识别功能调用的是 macOS 内置的实况文本引擎，也就是说：在图片上选中文字，右键复制，就能把文字提取出来。不联网，不上传，识别速度很快。

对经常需要从设计稿、UI 截图或者参考图里摘取文字的人来说，这个功能真的省事。

### 智能隐私打码

这是 Xnapper 里最"聪明"的功能。软件能自动识别截图中的邮箱地址、信用卡号、IP 地址、API 密钥等敏感信息，一键打码隐藏。

群里发截图、公开发教程，这个功能能替你过一遍，避免不小心泄露敏感内容。

### 截图历史记录

每次截图，Xnapper 都会保存原图和美化后的导出版本，还有对应的文字识别记录。回头想找某次截图？直接翻历史，不用重新截。

### 标注与社交媒体尺寸

支持添加箭头、文字、方框等标注元素。内置了 Twitter、LinkedIn、Instagram、YouTube 等主流社交平台的尺寸模板，一键切换比例，发哪个平台截图都不用手动裁剪。
